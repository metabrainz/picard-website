This quick start guide describes how you can tag your files using MusicBrainz Picard.

To make things clear - picard is not built to be a mass-single-track/recording tagger/tag fixer. Picard believes in quality over quantity and provides a plethora of customizations to tweak tags to your needs.

  1. Add your files using the Add Files or Add Folder toolbar buttons. For ease of use it is recommended to use the File Browser to manage file system interactions. Click on the View menu and then select File Browser
  2. Drag a folder to the Unmatched Files folder in the right-hand pane.
  3. Wait for Picard to process the files (the names will turn from grey to black) and then click the Cluster toolbar button to cluster files into album clusters.
  4. Select a clustered album and click the Lookup toolbar button to look the cluster up. Depending on your previous metadata, album will show up in the right-hand pane. If an album shows up, go to step 8. Since you are a new Picard user, we recommend that you read the steps in between as well.
  5. Alternatively, you can also try to use the Scan toolbar button to lookup files using their AcoustID, which is a unique finger-print associated with each recording.
  6. If nothing happened when you clicked the Lookup or Scan toolbar button, click the Lookup in Browser button toolbar button. Your default browser will open and allow you to find the right album manually.
  7. Pay attention to the amount of tracks in each album, and pick the one you have. Click the green TAGGER button.
  8. Drag the clustered album onto the album that was just opened, if it didn't happen automatically.

  9. Depending on your previous metadata, Picard will try to guess the matching tracks. The order is green > yellow > orange > red, where green is the best match. If you are seeing a lot of red and orange, it could mean that Picard has guessed wrong, or that your files didn't have a lot previous metadata to work with.

    If this is the case, it's recommended to click a track and compare the metadata using the Original Values and New Values at the bottom of picard window. If there is an incorrect match, simply drag & drop the track to its correct spot.

  10. When you are satisfied with the matches, select the album you want to save and click on the Save toolbar button to save files.

  11. A green check mark means the file is saved in its proper location.
