Setup and install guide for Picard under Linux. If you don't want to install third party binary packages, the source is available here.

Flatpak

Picard is available on Flathub. This version should work on all modern Linux distributions, as long Flatpak is installed.

First enable the Flathub repository: 

flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

You can now install Picard: 

flatpak install flathub org.musicbrainz.Picard
